# Emotion Detection using NLP & Machine Learning

This project implements emotion detection from text using Naive Bayes and Hugging Face Transformer models.  
The project was developed in Google Colab.

## How to Use
- Open the '.ipynb' file in Google Colab
- Run all cells
- Enter any custom text to get predicted emotion
